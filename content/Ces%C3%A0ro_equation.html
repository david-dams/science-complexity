Equation in geometry

In geometry, the Cesàro equation of a plane curve is an equation relating the curvature (κ) at a point of the curve to the arc length (s) from the start of the curve to the given point. It may also be given as an equation relating the radius of curvature (R) to arc length. (These are equivalent because R = ⁠1/κ⁠.) Two congruent curves will have the same Cesàro equation. Cesàro equations are named after Ernesto Cesàro.

Log-aesthetic curves

[edit]

The family of log-aesthetic curves[1] is determined in the general ( α ≠ 0 {\displaystyle \alpha \neq 0} ) case by the following intrinsic equation:

R ( s ) α = c 0 s + c 1 {\displaystyle R(s)^{\alpha }=c_{0}s+c_{1}}

This is equivalent to the following explicit formula for curvature:

κ ( s ) = ( c 0 s + c 1 ) − 1 / α {\displaystyle \kappa (s)=(c_{0}s+c_{1})^{-1/\alpha }}

Further, the c 1 {\displaystyle c_{1}} constant above represents simple re-parametrization of the arc length parameter, while c 0 {\displaystyle c_{0}} is equivalent to uniform scaling, so log-aesthetic curves are fully characterized by the α {\displaystyle \alpha } parameter.

In the special case of α = 0 {\displaystyle \alpha =0} , the log-aesthetic curve becomes Nielsen's spiral, with the following Cesàro equation (where a {\displaystyle a} is a uniform scaling parameter):

κ ( s ) = e s a a {\displaystyle \kappa (s)={\frac {e^{\frac {s}{a}}}{a}}}

A number of well known curves are instances of the log-aesthetic curve family. These include circle ( α = ∞ {\displaystyle \alpha =\infty } ), Euler spiral ( α = − 1 {\displaystyle \alpha =-1} ), Logarithmic spiral ( α = 1 {\displaystyle \alpha =1} ), and Circle involute ( α = 2 {\displaystyle \alpha =2} ).

Examples

[edit]

Some curves have a particularly simple representation by a Cesàro equation. Some examples are:

  • Line: κ = 0 {\displaystyle \kappa =0} .
  • Circle: κ = 1 α {\displaystyle \kappa ={\frac {1}{\alpha }}} , where α is the radius.
  • Logarithmic spiral: κ = C s {\displaystyle \kappa ={\frac {C}{s}}} , where C is a constant.
  • Circle involute: κ = C s {\displaystyle \kappa ={\frac {C}{\sqrt {s}}}} , where C is a constant.
  • Euler spiral: κ = C s {\displaystyle \kappa =Cs} , where C is a constant.
  • Catenary: κ = a s 2 + a 2 {\displaystyle \kappa ={\frac {a}{s^{2}+a^{2}}}} .

Related parameterizations

[edit]

The Cesàro equation of a curve is related to its Whewell equation in the following way: if the Whewell equation is φ = f (s) then the Cesàro equation is κ = f ′(s).
